1. Server Setup

The CaterBuddy server is built using Express.js and serves static files from the `frontend` folder.

2. File Structure

            /caterbuddy
            ├── frontend/
            │   ├── index.html
            │   ├── client.html
            │   ├── register.html
            │   ├── vendor.html
            │   ├── script.js
            │   ├── style.css
            │   ├── documentation.html
            │   ├── documentation.js
            │   ├── documentation.css
            ├── package.json
            ├── node_modules/

3. How to Run

            1. Install dependencies:
               npm install express
            2. Start the server:
               node index.js
            3. Open in browser:
               http://localhost:3000

4. Features

  • Serves static files from `frontend/`
  • Handles GET requests to `/` and serves `index.html`
  • Supports multiple frontend pages like client, vendor, and register
  • Runs on port 3000

5. API Endpoints

            GET /
              - Serves the frontend `index.html`
            
            Static Files:
              - The `frontend/` directory is exposed for serving static content.
